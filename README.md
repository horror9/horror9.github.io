# Rainy Neon Mail Clone 🌧️✨

ブラウザで動く。サーバー不要。  
localStorageにメール保存。  
コピーして `.html` にして開くだけ。

## デモ

※スクリーンショットを追加するとさらに雰囲気が出ます

## 特徴

- ネオンライトなUI  
- 雨のアニメーション  
- 完全クライアントサイド  
- localStorageで即保存  

## インストール不要

```bash
# そのままブラウザで開く
open index.html

使い方
	1.	ブラウザで開く
	2.	メール作成
	3.	送信（localStorageに保存）
	4.	受信（同じブラウザで閲覧）

注意
	•	サーバーなし
	•	別ブラウザ・別PCでは同期されない
	•	localStorageをクリアするとデータ消える
