タイトル：「収穫祭の影」

アメリカ西海岸、ロサンゼルス郊外。毎年恒例の「ヤシ収穫祭」が開かれ、観光客と地元民で街は賑わっていた。
祭りの目玉は、数十本のヤシの木を使った「ヤシの迷路」と「ヤシ飾りコンテスト」。

だが祭りの最中、突然一部のヤシが根ごと消える事件が発生。
防衛隊と地元警察は、群衆の目の前で行われたこの大胆な盗難に衝撃を受ける。

現場の監視カメラには、夜間に大型トラックが祭り会場へ侵入する様子が記録されていた。
トラックには祭り用に飾られたヤシが次々と積まれ、運び出されていく。

ドクター・リアム・コーエンは現場の土と葉のサンプルを採取。
分析の結果、過去に盗掘されたヤシと同じナノ粒子が付着していた。
「つまり、これは単なる祭りの事故ではなく、計画的な盗難だ」

さらに、防衛隊は群衆の目撃証言をもとに、犯人たちが祭りの装飾や照明を巧みに利用してトラックの侵入経路を隠していたことを突き止める。

狙いは、高級ヤシの希少性を維持するため、祭り用ヤシも市場価値を高めるために盗むという「都市象徴ビジネス」だった。
犯人は以前の密猟事件と同じ輸出業者で、祭りの賑わいを逆手に取った大胆な犯行だったのだ。

祭りの観客の一部は、自分の目の前でヤシが消える光景に言葉を失う。
記者のエマ・リヴァースは記事の最後にこう書き残す：
――祭りの喜びの影に、人間の欲望は静かに潜んでいた。

夜の会場を見下ろすハント大佐は低く呟く：
「人の群れの中でも、自然は守れない…欲望には敵わないのだ」
