# Neon Match-3

## 概要
**Neon Match-3** は、ネオンの雨が降る未来的な世界で遊べる、ブラウザ上のカジュアルパズルゲームです。  
プレイヤーはドラッグ操作でタイルをスライドさせ、同じ色のタイルを3つ以上並べて消すことを目指します。

ゲームは1ファイルで完結しており、コピーしてブラウザで開くだけで遊べます。

---

## 特徴

- 🌈 **ネオンの雨の下で遊ぶ幻想的な演出**
- 🎮 **ドラッグで操作可能**
- ✨ **やわらかいアニメーション**
- 💡 **タイルに影が付き、立体感のあるゲーム画面**
- 🔄 **ゲームっぽい挙動**（マッチ消去、連鎖、スコア計算など）
- 🖥 **ブラウザだけで完結**

---

## 遊び方

1. タイルをドラッグしてスライドさせる  
2. 同じ色のタイルを縦か横に3つ以上揃えると消える  
3. 消したタイルに応じてスコアが加算される  
4. 連鎖やコンボを狙って高スコアを目指そう

---

## 動作環境

- 最新のChrome、Firefox、Edgeなどのモダンブラウザ
- デスクトップ、タブレット、スマホのブラウザに対応

---

## インストール / 実行方法

1. このHTMLファイルをコピーする  
2. 任意のブラウザで開く  
3. そのままゲームスタート
