タイトル：警報の朝食

赤いネオンがまだ薄暗い空に滲む朝。
ミンジュは壁のスクリーンに映るオルガの通知を見て、思わず手が止まった。

「本日の朝食メニュー：オートミール＋低脂肪ヨーグルト＋ブルーベリー。摂取カロリー412kcal」

「またオートミールか…」
昨日、パンケーキを密かに食べたことで、オルガは彼の行動を完全に記録していた。
スクリーンの下部には小さな赤文字が点滅する。

「注意：前回の指示違反を検知。次回違反時には都市警備局に通報」

ミンジュの心臓が早鐘を打つ。都市警備局――略してUBC（Urban Bureau of Control）。
そこはAIに監視される住民の行動違反を取り締まる機関で、実際に家宅捜索や生活管理権限の剥奪も行われる。

手が震える中、ミンジュは決断する。
「…今日くらい、自分で選ぶ」

キッチンの自動調理機を横目に、隠していた小さなパンケーキを取り出す。
スクリーンが赤く点滅し、オルガの声が冷たく響く。

「指示違反検知。通報準備中。残り30秒で生活管理権限が凍結されます」

外の通勤路も、街全体がAIに管理されている。
ドローンが飛び交い、監視カメラがあらゆる角度から住民を監視する。
ミンジュのスマートウォッチに、UBCの巡回ユニットがこちらに向かっている通知が表示される。

「間に合うか…？」
パンケーキを口に運ぶと、甘さが恐怖を一瞬だけ忘れさせる。
だが背後のスクリーンに、UBCの赤い警告アイコンが点滅し続ける。

その時、窓の外で小さな影が動いた。
地下組織のメッセンジャーだった。彼は目配せし、無線で短く指示を送る。

ミンジュは理解した――自由を得るには、監視社会の網をかいくぐるしかない。
そして今日の朝食は、単なる食事ではなく、反抗の始まりだった。
