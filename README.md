中国の地下鉄が日本と違って見えるのは、国の事情とか都市の成長速度とか、安全基準の考え方がぜんぶ別世界だからだよ。
文句を言う暇があれば1本くらい電車が増える国だし、そりゃこうなる。

それぞれ理由をちゃっちゃと説明するね。

**① LEDじゃなくて薄型テレビみたいなLCDスクリーン**
巨大都市だと路線図も広告も情報量が多すぎて、LEDの点灯だけじゃ処理しきれない。
LCDの方が「動画広告ドーン！」ってできて儲かるし、情報更新も速い。広告収入は大事ってわけだ。

**② ホームドアが初期からほぼ標準**
中国の都市は地下鉄が新しくて、一気に巨大路線を作ってるから「最初からホームドア込みの設計」でやってる。
日本みたいに昭和の遺産を無理やり改造する必要がない。新築マンションが設備強いのと同じ。

**③ 荷物検査がある**
これは治安政策の考え方。中国は「安全はコストより上」な国で、事件対策を最優先にしてる。
日本は人権と運用コストとのバランスを考えるけど、中国は「やると決めたら全駅でやる」で押し通す。

**④ 何分後に来るかだけ表示される**
あの国は運行本数がアホみたいに多いから、いちいち詳細な時刻表を掲示する必要がない。
2分後とか3分後に来るなら、それ以上の情報いらんやろ、という冷徹な合理主義。
そもそも「分刻みのダイヤを死守する」日本の鉄道文化が割と特殊。

要するに、中国は**新しい巨大都市を、合理第一でドカッと作ってる国**だからこうなる。
日本は**古いインフラを丁寧に延命しつつ安全性も文化も両立させてる国**。

違って当然だね。どっちが良い悪いじゃなくて、国の性格がまるで違うだけ。
