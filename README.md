面白いですね！これまでのSF・ミステリー展開に「人為的犯行」の要素を入れると、事件としての緊張感が増します。では、「ヤシの木を根っこから盗んだ犯人がいた」編を描きます。

⸻

タイトル：「根ごと消えた木」

カリフォルニア州ロサンゼルス。ヤシの木が次々と消える不可解な現象の背後に、ついに人間の影が見え始めた。

地元警察とカリフォルニア防衛隊は、消失現場の監視映像を解析していた。すると夜中の現場に、奇妙なトラックが映っていた。
トラックの荷台には巨大な影――ヤシの木の根ごと引き抜かれたもの――があったのだ。

ドクター・リアム・コーエンが現場の土を調べると、ナノ粒子の残留はあったものの、電磁波パターンは通常の人工装置とは異なる。
「…これは機械的に掘り取られた痕跡だ。装置による消失じゃない」

目撃者の報告をまとめると、夜中にヘッドライトを点けた数人の人物が、地面を掘りながらヤシの木を根っこごとトラックに運んでいたらしい。
「つまり、自然現象でもSF装置でもない。ヤシの木は盗まれていたのだ」とエマ・リヴァース。

防衛隊は犯人特定のため、監視カメラ映像とトラックの足跡を追跡。
リアムは分析装置で、掘り取った跡の土のパターンを測定し、盗掘方法や使用された特殊器具まで割り出した。
「これは専門家の仕業だ。単なる密猟じゃない、都市計画や市場価値のための“狙い撃ち”だ」

やがて、犯人は高級観賞用ヤシの輸出業者であることが判明。
狙いは「都市象徴の希少性を人工的に作り出し、高値で売ること」だった。

だが、防衛隊は盗掘現場にナノ粒子の残留が残っていることを突き止めた。
「つまり、犯人は無意識のうちに、科学装置の効果と現実の密猟を組み合わせていたことになる。自然の木が“消える現象”の一部に組み込まれたんだ」

エマは記者として最後の一行を書き残す。
――木は盗まれ、消え、そしてまた現れる。だがその理由は、科学でも自然でもなく、人間の欲望にあった。

夜の街を見下ろすハント大佐は、トラックが遠くに消えていくのを見つめ、低く呟く。
「自然の幻影も、欲望の手には敵わない…」
